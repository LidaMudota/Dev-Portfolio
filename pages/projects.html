Мои проекты

Проект "Угадайка"

“Угадайка” — это интерактивная игра, в которой компьютер пытается угадать число, загаданное пользователем, используя подсказки “больше”, “меньше” и “верно”. Игра разработана с использованием HTML, CSS и JavaScript и имеет адаптивный дизайн для различных устройств. Технологии: HTML, CSS, JavaScript.


Проект "Калькулятор"

Этот проект представляет собой простой калькулятор, разработанный с использованием HTML, CSS и JavaScript. Он предоставляет базовые арифметические операции (сложение, вычитание, умножение, деление) и поддерживает ввод с клавиатуры. Проект также использует библиотеку Bootstrap для стилизации и шрифт Share Tech Mono для кнопок. Технологии: HTML, CSS, JavaScript.


Проект "WebTaskMaster"

Этот проект представляет собой веб-приложение, состоящее из нескольких задач (task), каждая из которых содержит свою HTML-страницу, CSS-стили и JavaScript-скрипты. Технологии: HTML, CSS, JavaScript.
